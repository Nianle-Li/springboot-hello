# SpringBoot Hello 项目
一个基础的 SpringBoot 入门项目，实现最简单的 HTTP 接口

## 功能
- 启动 SpringBoot 项目
- 访问 /hello 接口返回欢迎信息

## 技术栈
- SpringBoot 3.3.0
- Java 21

## 访问地址
启动项目后访问：
http://localhost:8080/hello
